# 🧹 Code Cleanup Guide - Files to Remove

This guide lists all the files and folders that should be removed as they are remnants from previous hackathons and development iterations.

## Files & Folders to Delete

### 1. **ILTD_temp/** (Entire Folder)
**Reason**: This is a complete backup/duplicate of the project from a previous hackathon phase
**What to do**: Delete the entire `ILTD_temp` folder

### 2. **Duplicate Vite Config Files** (in `DP world/` folder)
Delete these timestamp files:
- `vite.config.js.timestamp-1775217721186-8221d0abad291.mjs`
- `vite.config.js.timestamp-1775225668893-120d8aeede0a3.mjs`
- `vite.config.js.timestamp-1775272807684-560eeed06940a.mjs`

**Reason**: Auto-generated by Vite during development, not needed in version control

### 3. **Version Control/Diff Files** (Root directory)
Delete these files:
- `diff.txt`
- `diff_upstream.txt`
- `diff_upstream_va1.txt`

**Reason**: Temporary files used for comparing versions during hackathon

## How to Clean Up (Windows Command Prompt)

```batch
REM Delete ILTD_temp folder
rmdir /s /q ILTD_temp

REM Delete timestamp vite config files
del "DP world\vite.config.js.timestamp-1775217721186-8221d0abad291.mjs"
del "DP world\vite.config.js.timestamp-1775225668893-120d8aeede0a3.mjs"
del "DP world\vite.config.js.timestamp-1775272807684-560eeed06940a.mjs"

REM Delete diff files
del diff.txt
del diff_upstream.txt
del diff_upstream_va1.txt
```

## How to Clean Up (Mac/Linux Terminal)

```bash
# Delete ILTD_temp folder
rm -rf ILTD_temp

# Delete timestamp vite config files
rm "DP world/vite.config.js.timestamp-1775217721186-8221d0abad291.mjs"
rm "DP world/vite.config.js.timestamp-1775225668893-120d8aeede0a3.mjs"
rm "DP world/vite.config.js.timestamp-1775272807684-560eeed06940a.mjs"

# Delete diff files
rm diff.txt diff_upstream.txt diff_upstream_va1.txt
```
